Fixed Vertical Nav Bar

This navigation bar is vertical and fixed it does not move even when the page is scrolled, The contents scrolls but the navs remains in their positions...










Try to scroll this area, and see how the side nav sticks to the page

Notice that this div element has a left margin of 29%. This is because the side navigation is set to 28.5% width. If you remove the margin, the side nav will overlay/sit on top of this div.

Also notice that we have set overflow:auto to sidenav. This will add a scrollbar when the sidenav is too long (for example if it has over 50 links inside of it).
